# ⚔️ Dodge the Creeps - Godot 4.x (Port)

Este é o meu primeiro projeto oficial na **Godot Engine**. Ele foi desenvolvido como um desafio de aprendizado e migração de versão.

## 🚀 Sobre o Projeto
O projeto foi inspirado na playlist de Game Dev do canal **Clécio Espíndola**. 
- **O Desafio:** O tutorial original utiliza a **Godot 3**, e eu realizei a implementação e adaptação completa para a **Godot 4.x**, corrigindo sintaxes e funções legadas.
- **Engine:** Godot 4.x (GDScript).

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** GDScript.
- **Setup:** NVIDIA GeForce RTX 4060 | Ryzen 5 3600 | 24GB RAM.
- **Canal de Apoio:** [DragonSlayers](https://www.youtube.com/@DragonSlayers)

## 📁 Estrutura do Repositório
- `/scenes`: Ficheiros de cena (.tscn).
- `/scripts`: Lógica do jogo convertida para Godot 4 (.gd).
- `/dodge_the_creeps_2d_assets`: Arte e áudio.
